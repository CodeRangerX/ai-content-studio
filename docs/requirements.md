# 内容工坊 - 用户认证系统需求文档

## 1. 项目概述

### 1.1 背景
内容工坊是一个 AI 驱动的内容创作工具，目前为单页应用（SPA），无用户系统。为了提供个性化服务、保存用户历史记录、实现会员功能，需要开发用户认证系统。

### 1.2 目标
- 实现用户注册、登录、登出功能
- 支持邮箱验证
- 支持密码找回
- 为后续会员功能、历史记录等功能打下基础

---

## 2. 用户角色

| 角色 | 说明 | 权限 |
|------|------|------|
| 游客 | 未登录用户 | 使用基础生成功能（受限） |
| 注册用户 | 已登录用户 | 完整功能、保存历史记录 |
| VIP用户 | 付费用户 | 高级模板、更多生成次数 |

---

## 3. 功能需求

### 3.1 注册功能

#### 3.1.1 注册流程
```
用户填写邮箱 → 发送验证码 → 填写验证码 → 设置密码 → 注册成功
```

#### 3.1.2 输入项
- 邮箱（必填，格式验证）
- 密码（必填，8-20位，包含字母和数字）
- 确认密码（必填，与密码一致）

#### 3.1.3 验证规则
- 邮箱格式验证
- 邮箱唯一性检查
- 密码强度验证
- 验证码6位数字，有效期5分钟

### 3.2 登录功能

#### 3.2.1 登录流程
```
填写邮箱+密码 → 验证 → 登录成功 → 跳转首页
```

#### 3.2.2 输入项
- 邮箱（必填）
- 密码（必填）
- 记住我（可选，延长 Token 有效期）

#### 3.2.3 验证规则
- 邮箱格式验证
- 密码验证
- 错误次数限制（5次/小时，超出需验证码）
- 账户状态检查（是否已验证邮箱）

### 3.3 邮箱验证

#### 3.3.1 发送验证码
- 场景：注册、找回密码
- 频率限制：60秒内只能发送1次
- 内容：6位数字验证码
- 有效期：5分钟

#### 3.3.2 验证码邮件模板
```
主题：【内容工坊】验证码
内容：
您好！
您的验证码是：123456
验证码5分钟内有效，请勿泄露给他人。

— 内容工坊团队
```

### 3.4 密码找回

#### 3.4.1 找回流程
```
填写邮箱 → 发送验证码 → 验证邮箱 → 设置新密码 → 完成
```

#### 3.4.2 输入项
- 邮箱（必填）
- 验证码（必填）
- 新密码（必填）
- 确认新密码（必填）

### 3.5 登出功能
- 清除本地 Token
- 跳转到首页

### 3.6 用户状态管理

#### 3.6.1 Token 管理
- Access Token：有效期 2 小时
- Refresh Token：有效期 7 天（"记住我"为 30 天）
- 存储位置：localStorage / Cookie

#### 3.6.2 登录状态持久化
- 页面刷新保持登录状态
- Token 过期自动刷新
- 多端登录检测（可选）

## 4. 非功能需求

### 4.1 安全性
- 密码加密存储（bcrypt / argon2）
- HTTPS 传输
- CSRF 防护
- XSS 防护
- SQL 注入防护
- 验证码防暴力破解

### 4.2 性能
- 登录响应时间 < 500ms
- 验证码发送 < 3s
- 支持 100 并发用户

### 4.3 可用性
- 表单实时验证
- 友好的错误提示
- 支持中英文

### 4.4 兼容性
- 支持 Chrome、Firefox、Safari、Edge 最新版
- 支持移动端浏览器
- 支持 iOS Safari、Android Chrome

## 6. 数据需求

### 6.1 用户信息
| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 用户唯一标识 |
| email | string | 邮箱（唯一） |
| password_hash | string | 密码哈希 |
| nickname | string | 昵称（可选） |
| avatar | string | 头像 URL（可选） |
| email_verified | boolean | 邮箱是否验证 |
| created_at | timestamp | 注册时间 |
| updated_at | timestamp | 更新时间 |
| last_login_at | timestamp | 最后登录时间 |

### 6.2 验证码记录
| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 记录标识 |
| email | string | 邮箱 |
| code | string | 验证码 |
| type | string | 类型（register/reset_password） |
| expires_at | timestamp | 过期时间 |
| used | boolean | 是否已使用 |

## 8. 风险与应对

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|----------|
| 邮件发送失败 | 中 | 高 | 多服务商备份 |
| 暴力破解 | 中 | 高 | 限流 + 验证码 |
| 数据泄露 | 低 | 极高 | 加密存储 + HTTPS |
